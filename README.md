# Dimensionality Reduction and Clustering of Handwritten Digits using PCA and K-Means

## Objective
Apply PCA for dimensionality reduction and K-Means clustering on the Scikit-learn Digits Dataset.

## Tools Used
- Python
- Jupyter Notebook
- NumPy
- SciPy
- Scikit-learn

## Steps
1. Load Digits Dataset
2. Apply K-Means Clustering
3. Reduce dimensions using PCA
4. Cluster reduced data
5. Map clusters to digit labels
6. Evaluate using Accuracy, Confusion Matrix, and Classification Report
